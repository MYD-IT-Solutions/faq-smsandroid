Frequently Asked Questions

How to student promotion to next class in Zero ERP?
Student promotion to next class in Zero ERP. you have to select promotion in admission option then change school session and create class ,section. Fee details of every student can be stored in database through this module.
How to use zeroerp school android app dashboard .
New user can register in My school Android app they need to click on register button then enter a valid email ID. Check OTP for your email ID and then you need to enter OTP in verification page and then fill school information on the registration page.
How to add class and section in ZeroERP Android App?
Add new class and section in school mobile app please select class option and add class name and section name
How to create subject in ZeroERP Android App?
Add new subject in school mobile app please select create subject option and add new subject name
